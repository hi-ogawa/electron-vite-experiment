# electron-vite-experiment

Fiddling with electron and vite. Basic ideas are from https://github.com/cawa-93/vite-electron-builder.

```sh
# development
pnpm i
npm run dev

# package and run
npm run build
npm run package
./build/electron-vite-experiment-0.0.0.AppImage
```
